SAM Boilerplate — HTTP API CORS

CORS handled 100% by API Gateway. Lambda returns zero CORS headers.

Important This page must be served over HTTP(S), not file://. Browsers do not send the Origin header for file:// requests, so API Gateway cannot return CORS headers. Use python -m http.server 8080 in this directory and open http://localhost:8080/frontend.htm.
